
<h1 style="text-align: center">RuoYi-Antd v1.0</h1>
<h4 style="text-align: center">基于React+Antd的若依Web前端快速框架</h4>

## 平台简介
* 本仓库为基于若依的React + UmiJS + Antd的Web前端，后端请使用若依RuoYi-Vue
* 前端技术栈：
  * React
  * TypeScript
  * Pnpm
  * UmiJS
  * Ant Design
  * Ant Design Pro
  * TailWindCss

## 运行环境
* node v22.6.0

## 构建工具
* pnpm

## 安装、运行
```angular2html
# 克隆项目
# 进入项目目录
cd RuoYi-Antd

# 安装依赖
pnpm install

# 启动服务
pnpm run dev

# 构建
pnpm run build
```
